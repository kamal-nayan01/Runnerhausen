# Runnerhausen

## Detailed Project Information

### Features
- Engaging gameplay with multiple levels and challenges.
- Stunning graphics and a captivating soundtrack.
- Support for multiplayer and single-player modes.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/kamal-nayan01/runnerhausen.git
   ```
2. Navigate to the project directory:
   ```bash
   cd runnerhausen
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

### Gameplay Mechanics
- The player navigates through various obstacles to reach the finish line.
- Collect power-ups to enhance gameplay.
- Compete against the clock for a better score.

### Controls
- Move Left: `A`
- Move Right: `D`
- Jump: `Space`
- Pause: `P`

### Deployment Instructions
- To deploy the project, ensure all configurations are set in the `config.js` file. 
- Use the following command to start the server:
   ```bash
   npm start
   ```
